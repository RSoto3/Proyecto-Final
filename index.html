Tipos de familia

La ID de cada Familia es el número que lo acompaña.

Tipo 1: Adulto 1
Tipo 2: Adulto 2
Tipo 3: Adulto 1 - Adulto 2
Tipo 4: Adulto 1 - Adulto 2 - Infante 1
Tipo 5: Adulto 1 - Adulto 2 - Infante 1 - Infante 2
Tipo 6: Adulto 1 - Adulto 2 - Infante 1 - Infante 2 - Infante 3

Compañias Disponibles

1)Hospital
2)Bomberos
3)Policia

Indicaciones.

1) Para visualizar el grafo aleatorio debe clickear en crear.
2) Para Mostrar el camino mas corto debe ingresar un nodo inicial y un nodo final, ambas con sus respectivas ID.
3) Para Mostrar un incendio, debe elegir una familia que se encuentre en el grafo, ingrese la ID que la identifica en la casilla "INCENDIO EN".
4) Para mostrar un Robo, debe elegir una familia que se encuentre en el grafo, ingrese la ID que la identifica en la casilla "ROBO EN"
5) Para mostrar una emergencia, debe elegir una familia que se encuentre en el grafo, ingrese la ID que la identifica en la casilla "EMERGENCIA MEDICA EN"

Restricciones

1) No se pueden colocar mas de una ID por casilla, si desea cambiar de ID, esta debera reemplazar la anterior.
2) La unica opción para interactuar entre compañias es "Mostrar camino mas corto".
